Uninstalling the Edge for Private Cloud Developer Services portal

Developer Services Portal v. 4.17.09

To uninstall the Edge for Private Cloud Developer Services portal completely from your system, perform the steps described in http://docs.apigee.com/node/20406. 

Alternatively, you can uninstall the following Developer Services portal components individually:

  • apigee-drupal
  • apigee-drupal-contrib
  • apigee-drupal-devportal
  • apigee-drush
  • apigee-php

For complete details, see http://docs.apigee.com/node/20406.
